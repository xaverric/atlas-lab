Design Decisions Summary

Layout

Sidebar tree + content table (option A)

Sidebar Behavior

Collapsible via hamburger button, hidden by default on mobile

Tree Content

Full tree: folders AND individual items (notes/files)

Tree Metadata

Name + item count + total size per folder

Visibility Indicators

Colored left border (blue=public, gray=private) + icon badge with permission level

Actions

Inline icons (preview, download) + three-dot menu + right-click context menu

Notes Columns

Name, Size, Attachments (count + size), Visibility, Author, Updated

Files Columns

Name, Type, Size, Tags, Visibility, Uploaded
