Overview

The Atlas API uses JWT tokens issued by Keycloak for authentication. All protected endpoints require a valid access token in the Authorization header.

Key Design Elements

Editor

TipTap WYSIWYG with markdown backend. Toolbar: bold, italic, strike, code, headings, lists, tasks, links, images, code blocks. "Markdown" toggle to see raw source.

Code Blocks

CodeMirror inside TipTap for code blocks. Language selector. Same CodeMirror used in scheduler.

Images

DMS attachments inline with "DMS Attachment" badge + "Open in Files" link. External URLs as regular images.

Attachments

List below content. Each has: icon, name, size, "Open in Files" (opens detail modal), remove button. Inline images also appear here.

History Drawer

Right panel, toggled via History button. Shows all saved versions with: author, timestamp, summary. "Restore" button per version.

Scheduler

CodeMirror replaces the plain <pre> block in job config display. Same theme as notes code blocks.
