Overview

The Atlas API uses JWT tokens issued by Keycloak for authentication. All protected endpoints require a valid access token in the Authorization header.

Setup

First, configure your environment variables:

javascript Copy 
1const KEYCLOAK_URL = 'http://auth.example.com';
2const REALM = 'atlas';
3const CLIENT_ID = 'atlas-gui';
4
5// Initialize the OIDC client
6const settings = {
7  authority: `${KEYCLOAK_URL}/realms/${REALM}`,
8  client_id: CLIENT_ID,
9  redirect_uri: 'http://localhost:3000/callback',
10};

The token is automatically attached to API requests:
